# SpringBoot-Vue-030_抗疫物资管理系统

#### Description
基于SpringBoot+MyBatisPlus+Shiro的抗疫物资管理系统，实现物资全生命周期管理。系统包含物资分类管理、出入库管理、公告发布、用户权限控制等功能，采用分层架构设计，支持高效数据操作和权限控制。
